# API-Iskdeamon
Как запустить проект
Запустить Docker контейнер iskdaemon
(он должен работать и слушать порт 31128)

Запусти PHP сервер
cd "Перейти в папку прокета"
php -S 127.0.0.1:8080 -t public

Открой тестовую страницу
http://127.0.0.1:8080

Проверь API
нажать кнопку “Проверить API”

Добавляй картинки
"Добавить в индекс" - добавляет картинку в базу
"Поиск по картинке" - ищет похожие картинки
